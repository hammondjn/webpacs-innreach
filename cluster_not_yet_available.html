What is a Cluster?

Generally speaking, a cluster is an organizational group of Missouri libraries, within the MOBIUS library consortium. Specifically, the implementation plan of the Common Library Platform calls for 11 automated library systems linked together through a separate union catalog system. A system is defined as a single server running library software from Innovative Interfaces, Inc. Each of the 11 library systems will be shared by multiple institutions. The institutions that share a system are called a cluster with proximity as the primary factor for determining how the clusters are formed. [more]
